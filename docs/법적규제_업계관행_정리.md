### 1-2. 개인정보보호법

#### 도면 이미지 처리

- 아파트 도면 이미지 = **주소 식별 가능 개인정보**로 분류
- 수집 전 명시적 동의 필수 (회원가입 없이 업로드 허용 시 → **업로드 전 동의 UI 필수**)

**비회원 서비스의 경우 — 업로드 직전 동의 처리:**

```
방법: 업로드 버튼 위 체크박스 또는 최초 1회 모달
내용 (개인정보보호법 제15조 필수 4항목):
  1. 수집 항목: 도면 이미지
  2. 수집 목적: 공간 구조 분석 및 인테리어 추천 서비스 제공
  3. 보유 기간: 분석 완료 후 30일 이내 자동 삭제
  4. 동의 거부 권리 및 불이익: 거부 시 서비스 이용 불가
```

> **중요:** 4가지 항목 중 하나라도 빠지면 유효한 동의로 인정되지 않음

#### 도면 데이터 AI 학습 활용 여부

| 용도 | 동의 필요 여부 |
|------|--------------|
| 서비스 제공 (분석 → 추천) | 약관 동의 1회로 충분 |
| **AI 모델 재학습 데이터로 활용** | **별도 명시적 동의 필요** (분리 체크박스) |

> Claude API / Gemini API 사용 시: 기본 설정에서 API 입력이 학습에 사용되지 않으므로 별도 동의 절차 불필요

### 1-3. 건축사법 / 건설산업기본법

| 항목 | 내용 |
|------|------|
| 실내건축공사업 면허 | 공사예정금액 **1,500만원 이상** 시공 중개 시 건설업 면허 필수 |
| 등록 요건 | 자본금 1.5억 / 건설기술자 2인 이상 / 독립 사무실 |

| 현재 규제 공백 | 민간 AI 건축 분석 서비스에 대한 구체적 규정 미정립 (공백 ≠ 면책) |

**대응 방법:** "참고·추정" 표현 사용 + 면책 문구 명시

### 1-5. 전자상거래법 / 표시광고법

- 네이버쇼핑·한샘·일룸 상품 링크 연결 = **전자상거래**로 분류
- 가격·재고 오류 발생 시 플랫폼 책임 범위 약관에 명시 필요
- **"플랫폼은 일체의 책임을 지지 않는다"** → 공정위 적발 대상 (불공정 약관)

## 2. 업계 관행

### 2-1. 공정거래위원회 시정 사례 (2024년 12월)

공정위가 오늘의집·숨고·집닥 등 인테리어 플랫폼 6개사 약관 **84개 조항** 시정.

| 플랫폼 | 기존 불공정 약관 | 시정 내용 |
|-------|----------------|---------|
| 집닥 | "시공 책임은 전문가에만 있음, 플랫폼 무책임" | 고의·중과실 범위 내 플랫폼도 책임 |
| 오늘의집 | "이용자 귀책사유 손해 책임 없음" | 플랫폼 고의·중과실에 대한 책임 부담 |
| 숨고 | "게시물 사전통지 없이 임시조치 가능" | 삭제 전 사용자 통지, 이의제기 절차 마련 |
| 집꾸미기 | "회원 게시물 자유 이용·복제·배포 가능" | 서비스 제공 필요 범위로 제한 |

> **핵심 원칙: 절대 면책 불가. 고의·중과실에 대한 책임은 반드시 부담해야 함.**

### 2-2. 렌더링 이미지 업계 표준 고지 방식

오늘의집·집닥 등 주요 플랫폼의 실무 고지 방식:

```
"본 렌더링 이미지는 생성형 AI 기술로 제작되었으며,
 - 실제 시공 결과와 색감·질감·조명이 다를 수 있습니다.
 - 설치된 제품의 규격 오류 발견 시 사용자 책임입니다.
 - 최종 결정 전 샘플·현장 확인을 권장합니다."
```

### 2-3. 인테리어 업체 검증 관행

| 항목 | 현실 |
|------|------|
| 무등록 업체 | 1,500만원 미만 공사는 면허 없이 가능 → 플랫폼에 대량 입점 |
| 플랫폼 심사 기준 | 입점 수수료·리뷰·중개 실적 (건설업 등록 확인 없음이 관행) |

| 피해 유형 | 시공품질 불량 62.1% / 계약불이행·하자보수 불충분 |

## 3. 사용자 기대치 & 분쟁 사례

### 3-1. 실제 분쟁 원인 TOP 3

1. **렌더링 과대 표현** — 조명·색감·재질이 실제와 달라 "사기당했다" 인식
2. **비용 추정 오차** — 공사 범위 불명확 + 추가 공사(배관·전기) 발생
3. **책임 소재 불명확** — "AI가 권장했으니 플랫폼 책임" vs "참고용이므로 사용자 책임"

### 3-2. 수용 가능한 오차 범위 (업계 기준)

| 항목 | 수용 범위 | 조건 |
|------|----------|------|
| 비용 추정 오차 | **±20~30%** | 추가 비용 발생 조건 사전 안내 시 분쟁 감소 |
| 렌더링 vs 실제 차이 | 사전 "참고용" 안내 시 수용 | 안내 없으면 분쟁 |
| AI 구조 분석 신뢰도 | 신뢰도 % 함께 표시 시 수용 | 단정적 표현 → 분쟁 |

## 4. 필수 면책 체크리스트

서비스 약관 및 UI에 반드시 포함해야 하는 항목.

### ✅ 도면 이미지 처리
```
- 사용자 업로드 동의 취득 (업로드 전 체크박스 또는 최초 1회 모달)
- "도면은 분석 완료 후 30일 이내 자동 삭제"
- "도면 정보는 제3자에게 공유되지 않습니다"
```

### ✅ AI 구조 분석 (내력벽 판별)
```
- "벽 분류는 도면 두께·패턴 기반 AI 추정입니다 (신뢰도 70~85%)"
- "실제 시공 전 관리사무소·구조 기술사 확인이 필수입니다"
- "분양 카탈로그 도면은 구조 도면이 아니므로 오차가 있을 수 있습니다"
- "본 서비스는 건축설계 서비스가 아닙니다"
```

### ✅ 렌더링 이미지 (AI 기본법 의무)
```
- "이 이미지는 Gemini Imagen(생성형 AI)으로 제작되었습니다" (화면 표기 필수)
- "실제 시공 결과와 색감·질감·규격이 다를 수 있습니다"
- 워터마크 또는 메타데이터 삽입
```

### ✅ 상품 링크 연결
```
- "가격·재고는 검색 시점 기준이며 실시간 반영이 아닐 수 있습니다"
- "상품 구매 후 반품·환불은 해당 판매처 규정을 따릅니다"
```

### ✅ 비용 추정
```
- "AI 기반 참고 단가이며 실제 공사비는 ±30% 오차 발생 가능"
- "배관·전기 등 현장 상황에 따라 추가 비용 발생 가능"
- "최종 견적은 시공업체 직접 확인 필수"
```

### ✅ SNS 이미지 업로드
```
- "저작권 침해 콘텐츠 업로드 시 책임은 사용자에게 있습니다"
- 저작권 침해 신고 및 삭제 처리 절차 명시
```

---

❌ "우리 AI는 100% 정확합니다"
