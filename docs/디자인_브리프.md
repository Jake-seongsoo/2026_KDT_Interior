## 1. 누구를 위해 — 핵심 페르소나

**이서연, 31세 · 첫 내 집을 꾸미는 신혼 1년 차**
> *"오늘의집 300장 저장했는데, 정작 우리 집엔 뭘 해야 할지 모르겠어요. 누가 '이거면 됐다'고 한 번만 정리해줬으면."*

| 축 | 내용 |
|----|------|
| 숙련도 | **거의 0 — 구조적 초보.** 인테리어는 일생 1~2번뿐이라 경험이 안 쌓임 |
| 사용 빈도 | **극저빈도(이벤트성).** 이사 한 번에 집중 → 입주 후 종료 |
| 시간 압박 | **중~높음.** 입주 일정에 묶이면서도 후회 비용 탓에 결정을 미루는 모순 |
| 사용 환경 | **혼자 시작하지만 혼자 결정 못 함** — 배우자·지인에게 검증 위임("결정의 사회화") |
| 기기 | **모바일 우선** — 입력원이 네이버 부동산 폰 캡처, 결과는 배우자 폰으로 공유 |

진짜 통증은 "예쁜 시안"이 아니라 **"틀리지 않았다는 확신"과 "예산 통제감"**.
유사 사례: 셀프 강행 신혼부부 예산 4,000만→6,000만·10개월 임시거주(머니투데이), "첫 집 인테리어 망하는 법"(Threads) — 통증이 현실 리스크임을 증명.

## 2. 무슨 일을 끝내려는가 — JTBD

> **언제** 첫 집 잔금을 치르고 입주는 다가오는데 오늘의집을 수백 장 봐도 "우리 집 버전"이 안 그려질 때,
> **나는** 내 도면·예산·취향에 맞는 방향을 한 번에 정리받아,
> **그래서** 불안과 결정 피로를 끊고 "이거면 됐다"고 확신하며 다음 단계로 넘어가고 싶다.

| 층 | 끝내려는 일 | 지배 감정 |
|----|------------|----------|
| 기능적 | 내 공간에 맞는 방향(톤·방별 모습)을 본다 / 예산 적정성을 판단한다 | 막막함 · 통제감 상실 |
| 정서적 | "이게 맞나" 불안을 잠재우고, 결정을 빨리 매듭짓는다 | 불안 · 피로 |
| 사회적 | 배우자·지인에게 검증받아 "혼자 잘못 정한 게 아님"을 확인한다 | 외로움 · 책임 분산 욕구 |

**일이 끝난 신호:** "이 방향이면 됐다"며 떠남 + **공유**해서 "괜찮은데?"를 돌려받음.
**안 끝난 신호:** 톤을 못 좁힘 / 예산 근거 없음 / 공유해도 의견 못 받음 → 다시 핀터레스트로.

| 단계 | 도구 | 시간 | 이탈 |

| ② 콘셉트 결정 | 머릿속·캡처 모음 | 🟡 | **포기 A — 여기서 자주 멈춤** |
| ③ 도면·배치 시뮬 | 오늘의집 3D·줄자·Planner 5D | 🟡 | 실측 오차·2D 한계 |

### 감정 지도 — 부정 감정의 정점

```
 高 │                                              ▲ 피크2 (계약·시공)
    │                          ▲ 피크1            ╱  불안·사기공포·후회
    │            ┌─ 자기의심  ╱ (견적비교)       ╱
    │  설렘     ╱ 막막·피로 ╱ 답답·불신         ╱
 低 │ ╱────────╱──────────╱───────────────────╱········ (사후) 박탈감·우울
    └──①──────②──────③────④──────⑤──────⑥──────⑦──────▶
```

| 표면 불만 | **진짜 원인** |
|----------|-------------|
| 영감이 너무 많다 | 비교 기준을 못 세움 (판단 축 부재) |
| 의견이 안 맞는다 | 객관 근거 없어 '취향 대결'이 됨 |
| 견적이 헷갈린다 | 정보 비대칭 + 적정가 기준 없음 → 통제감 상실 |
| 업체를 못 믿겠다 | 검증 수단 없는데 비용은 비가역·고액 |
| SNS가 너무 예쁘다 | 연출 이미지가 기준점 + 내 집과 구조 갭 |

> **핵심:** 표면 불만은 다양하지만 진짜 원인은 한 곳으로 수렴한다 — **"내 선택이 맞다고 확인해줄 기준·근거의 부재."**

## 4. 장르 표준 vs 우리만의 것 — UX 벤치마크

조사 대상: RoomGPT/RoomsGPT, Interior AI, Spacely AI, Spacejoy, Collov AI.
**장르 표준 흐름:** 사진 업로드 → 방 유형 → 스타일 선택 → 생성 → Before/After → Variations → Shop the look → 저장/공유.

### A. 그대로 따라야 할 표준 (Moodie 이미 대부분 따름)

직선형 플로우, 카드 그리드 스타일 선택(`ToneCandidateGrid`), 방 유형 탭(`RoomTabs`), Shop the look(`ProductGrid`), 커스텀 프롬프트(직접 입력 모드), 로딩 표시, 저장·공유, AI 생성 고지.
→ **단, "여러 안 비교(variations)"는 표준인데 Moodie가 약함**(미스핏 A). "톤" 용어도 첫 화면에서 한 줄 정의 권장.

### B. Moodie에만 있어 **새로 안내**해야 할 것 (경쟁사에 없음 = 기대 밖)

| # | 고유 | 안내 필요 이유 |
|---|------|--------------|
| B1 | 입력 = **도면(평면도)**, 방 사진 아님 | 사용자가 방 사진을 올리려 함 → 예시 도면 이미지로 강화 |
| B2 | **Before 없음**(빈 집 전제) | Before/After 기대를 프레이밍으로 차단 |
| B3 | **집 전체를 방별로** 한 번에 | "전체를 방별로 제안" 안내 + 배치도 조망 |
| B4 | **근거·팔레트 붙은 톤 6개** | 개인화가 차별점인데 지금 잘려 안 보임(→ A) |
| B5 | **2D SVG 배치도** | 상호작용 힌트 필요 |
| B6 | **정밀화(예산·가족·가전)** | "정밀화"는 비표준 용어 → 풀어쓰기 |

## 5. 성공의 정의 — 관찰 가능한 한 행동

> **"사용자가 하나의 결과를 골라 공유한다."**

| 증명하는 것 | 근거 |
|------------|------|
| 결정을 닫았다 | "내 답"이라 굳혀야 남에게 보냄 |
| 검증 동기 충족 | 1순위 동기 = 사회적 검증 |
| 포트폴리오 확산 | 공유 = 지인 베타 |

- **관측:** `handleShare` 클릭 (`result/[id]/page.tsx`).
- **지표:** 결과 도달자 중 1회 이상 공유 비율. 결과 도달·상품 클릭은 수동적 퍼널 통과일 뿐 "확신"을 증명하지 못함.

## 6. 디자인 원칙 (리서치에서 도출)

1. **확신을 만든다, 선택지를 늘리지 않는다.** 사용자의 결핍은 영감이 아니라 판단 기준 → 더 많은 옵션은 결정 피로를 키운다.
2. **결과 화면은 "보여줘라"가 가장 크게 들린다.** "더 탐색하라"(다른 톤·정밀화)가 아니라 "이게 네 답이다, 공유하라".
3. **근거를 보여 확신을 떠받친다.** "왜 이 톤인지"·"내 도면 기반"을 잘리지 않게 노출 → 자기의심(②)·박탈감(사후) 완화.
4. **모바일·초보 기준으로 안내한다.** 도면 입력(사진 아님)·평형·배치도 등 우리만의 것은 명시적 온보딩.
5. **제거가 아니라 비중 조정.** 핵심 기능(F006·F007)은 숨기지 않고 시각 우선순위만 낮춘다.

## 7. 화면 우선순위 — 미스핏 백로그 + 키울 것/물릴 것

### 미스핏 (영향 큰 순)

| 미스핏 | 막히는 것 | 연결 |
|--------|----------|------|
| 🔴 **A. 톤 비교·검증 단방향** | 결정 피로 종료 + 사회적 검증 | 포기 A, 감정 ②⑥, 표준 variations |
| 🔴 **B. 예산 신호 없음** | 예산 적정성 판단 | 포기 B, 감정 ⑤ |
| 🟡 **C. Vision 오인식 수정 불가** | 내 공간 정확 반영 | 신뢰, B1·B4 |
| 🟡 **D. 모바일·데스크톱형 인터랙션** | 입력·공유 편의 | 모바일 페르소나, PDF=print |

### 결과 화면: 키울 것 / 물릴 것

**🔼 키울 것**
1. **공유를 1순위 primary CTA로 격상** — 지금 5개 버튼 동일 비중에서 묻힘. 동기형 라벨("배우자에게 보내기").
2. **선택 톤의 확신 근거 키우기** — `line-clamp` 풀어 "왜 이 톤인지" 노출.
3. **"내 답" 요약 한눈에** — 선택 톤 + 방별 대표 1장 + 팔레트, 공유 미리보기/OG도 자랑스럽게.

**🔽 뒤로 물릴 것**
1. **재탐색 동선(다른 톤·정밀화) secondary로** — "아직 확신 못 함" 신호라 결정을 미루게 함.
2. **PDF·홈으로 부가 동선으로** — PDF는 모바일 약함, 공유에 자리 양보.
3. **상품 그리드 점유를 결정 이후로** — 구매는 결정을 닫은 다음 행동.
