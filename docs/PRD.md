# Moodie · AI 인테리어 추천 서비스 MVP PRD

## 핵심 정보

**목적**: 도면 이미지 업로드만으로 Claude Vision이 공간을 분석하고 2026 트렌드 기반 톤 6개를 제안한다. 사용자가 톤 1개를 선택하면 방별 SVG 2D 배치도 + Imagen 렌더링 + 실구매 상품을 자동 생성해, 생애 첫 집 인테리어를 혼자 결정하는 스트레스를 제거한다.

**사용자**: 전문 업체 없이 생애 첫 집 인테리어를 혼자 결정해야 하는 20~30대 신혼부부 및 1인 가구.

---

## 사용자 여정

```
1. 랜딩 페이지
   ↓ "도면 분석 시작" 버튼 클릭

2. 로그인 여부 확인
   ├── 비로그인 → 로그인 필요 안내 팝업 → 로그인 페이지
   └── 로그인   → 도면 업로드 및 AI 분석 진행

3. 도면 업로드 페이지
   ↓ 모드 선택 (AI 자동 추천 / 직접 입력) + JPG/PNG 파일 선택 + 공급면적(평) 입력
   ├── [AI 자동 추천] "분석 시작" 버튼 클릭
   └── [직접 입력]   자유 텍스트 + 무드 칩 입력 후 "분석 시작" 버튼 클릭

4. AI 분석 대기 페이지 (자동)
   ├── [AI 자동] Claude Vision 공간 분석 + Web Search 트렌드 기반 톤 후보 6개 생성 완료 (약 10~15초)
   └── [직접 입력] Claude Vision 공간 분석 + 사용자 입력 기반 톤 3개 변형 생성 완료 (약 10~15초)

5. 분석 결과 & 톤 선택 페이지
   ├── [AI 자동] 톤 카드 6개 중 1개 선택 → "이 톤으로 시안 만들기" 버튼 클릭
   └── [직접 입력] 톤 카드 3개(안전·중립·대담) 중 1개 선택 → "이 톤으로 시안 만들기" 버튼 클릭
   └── 방 정보가 틀렸을 경우 → 직접 수정 후 재분석

6. 방별 시안 생성 대기 페이지 (자동)
   ↓ SVG 2D 배치도 + Vertex AI Imagen 4 방별 렌더링 + 이케아 상품 검색 병렬 완료 (약 20~40초)

7. 방별 시안 결과 페이지 (핵심 페이지)
   ↓ SVG 배치도 확인 → 방 탭 전환으로 각 방 렌더링·추천 근거·상품 확인

   [톤 재선택]  → 분석 결과 & 톤 선택 페이지로 복귀 (방 분석 재사용, 새 톤 선택)
   [정밀화]     → 정밀화 입력 모달(예산/가족형태/취향) → 같은 톤으로 재렌더링 → 방별 시안 결과 페이지
8. 결과 저장 / 공유
   ↓ "공유 링크 복사" 또는 "PDF 저장" 클릭

9. 완료 → 재업로드(새 도면 분석) 또는 내 분석 기록 페이지 이동
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F001** | 도면 이미지 업로드 | JPG/PNG 5MB 이내 파일 업로드, 미리보기 표시 | 서비스 시작점, 없으면 전체 파이프라인 불가 | 도면 업로드 페이지 |
| **F002** | Claude Vision 공간 분석 | 방 개수·용도·비율·문·창문 자동 추출, 도면 오버레이 표시 | 핵심 AI 가치 제공, 포트폴리오 기술 시연 핵심 | AI 분석 대기 페이지, 분석 결과 확인 페이지 |
| **F003** | 분석 결과 수동 수정 | Claude Vision 추출 정보 텍스트 직접 수정 | 80% 정확도 한계 보완, 사용자 신뢰 확보 | 분석 결과 확인 페이지 |
| **F004** | AI 자동 톤 6개 생성 | Claude Vision 공간 분석과 동시에 Web Search로 2026 트렌드 검색 → 공간 특성에 맞는 톤 후보 6개(이름·설명·추천 근거) 생성 | AI 기반 개인화 출발점, 선택 UX로 사용자 관여도 향상 | AI 분석 대기 페이지, 분석 결과 & 톤 선택 페이지 |
| **F004-C** | 사용자 입력 기반 톤 3개 변형 생성 | 업로드 페이지 "직접 입력" 모드에서 자유 텍스트(최대 500자) + 무드 칩(사전 12개 + 자유 태그)으로 원하는 분위기 입력 → Claude가 안전·중립·대담 3가지 해석 변형 생성. POST /analyze/custom. 트렌드 캐시를 F004와 공유. | 톤 선택의 사용자 주도성 확보, AI 추천만으로 부족한 사용자 대응 | 도면 업로드 페이지(모드 선택), AI 분석 대기 페이지, 분석 결과 & 톤 선택 페이지 |
| **F004-B** | 방별 시안 자동 생성 | 선택된 톤 기반으로 주요 방 최대 4개 병렬 처리 — SVG 2D 탑뷰 배치도 + Vertex AI Imagen 4 방별 렌더링 이미지 + Claude 추천 근거 텍스트 | 서비스 핵심 결과물, AI 파이프라인 완성 | 방별 시안 생성 대기 페이지, 방별 시안 결과 페이지 |
| **F005** | 실구매 상품 추천 | 이케아 비공식 API 실시간 검색, 슬롯당 5개, Vision 기반 재랭킹 적용, 각 방별 상품 목록 (상품명·가격·구매 링크). 이케아 공식 사이트에서 가격·재고 직접 확인 안내 필수. | 인테리어 결정에서 구매까지 연결, 실용 가치 제공 | 방별 시안 결과 페이지 |
| **F006** | 톤 재선택 | 결과 페이지에서 "다시 톤 선택" 클릭 시 분석 결과 & 톤 선택 페이지로 복귀, 방 분석 재사용해 새 톤으로 재렌더링 | 비교를 통한 만족도 향상 | 방별 시안 결과 페이지, 분석 결과 & 톤 선택 페이지 |
| **F007** | 정밀화 맞춤 렌더링 | 예산·가족형태·취향·신혼 가전 배치(10종 체크리스트 + 방 지정) 입력 모달 → 현재 선택된 톤 유지 + 정밀화 파라미터로 방별 재렌더링. 방별 "예상 배치 가전" 라벨 + 면책 문구 노출. | 개인화로 결과 품질 향상 | 방별 시안 결과 페이지 (정밀화 입력 모달) |
| **F008** | 공유 링크 & PDF 저장 | 결과 페이지 공유 링크 생성, PDF 다운로드 | 지인 베타 확산, 포트폴리오 활용 | 방별 시안 결과 페이지 |
| **F009** | 좋아요/싫어요 피드백 | 각 방별 시안에 대한 👍/👎 반응 수집 | 재렌더링 품질 개선 데이터 확보 | 방별 시안 결과 페이지 |

## 기술 스택

> 아래는 실제 채택·구현된 스택 기준. 초기 설계안(Vite, SQLAlchemy, AWS S3 등)과 다름.

### 프론트엔드

- **Next.js 15** (App Router) — SSR·CSR 혼합, `/share/[uuid]` SSR용
- **React 19** — UI 라이브러리
- **TypeScript** — 타입 안전성 보장
- **Tailwind CSS v4** — 설정 파일 없는 CSS 엔진
- **shadcn/ui** — Dialog, Label 등 컴포넌트 사용 (components.json 정식 도입)
- **Vercel** — 자동 배포

### 백엔드

- **FastAPI** (Python 3.12) — 비동기 REST API 서버
- **Pydantic v2** — 요청/응답 스키마 검증
- **GCP Cloud Run** — 컨테이너 배포, min-instances=0 (발표 당일만 1)

### 데이터베이스 / 인증 / 스토리지

- **Supabase** (PostgreSQL 무료 500MB) — 관계형 DB + RLS
- **Supabase Auth** — Google OAuth 2.0, JWT 발급
- **GCP Cloud Storage** — 도면(`floor-plans/`) Signed URL 15분, 렌더링(`renders/`) 30일 Object Lifecycle 자동 삭제

### AI / 외부 API

- **Claude Sonnet 4.6** (Vision + Web Search) — 도면 공간 분석, 톤 후보 생성, 가구 검색어 생성
- **Vertex AI Imagen 4** (`imagen-4.0-generate-001`) — 방별 인테리어 렌더링 이미지 생성
- **이케아 비공식 API** (`sik.search.blue.cdtapps.com`) — 실구매 가구 검색, API 키 불필요
- **svgwrite** — 2D 탑뷰 배치도 SVG 생성
- **cachetools TTLCache** — 트렌드 검색 결과 24h 메모리 캐시
