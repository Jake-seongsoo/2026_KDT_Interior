**증상:** 주요 방 4장 중 1장이 Vertex AI 503/429 반환 시 전체 추천 실패
**위치:** `backend/services/imagen_service.py`
```python
# 위험한 코드
results = await asyncio.gather(imagen_a(), imagen_b(), imagen_c())

# 안전한 코드
results = await asyncio.gather(
  render_livingroom(), render_kitchen(), render_bedroom(), render_room(),
  return_exceptions=True

# 실패한 항목은 fallback 이미지 또는 재시도 처리
```

### [RISK-02] GCS 도면 이미지 직접 접근 가능 (개인정보 노출)
**증상:** Cloud Storage public URL 알면 타인 도면 열람 가능
**위치:** `backend/services/storage_service.py`
```python
# 위험: public URL 반환
return f'https://storage.googleapis.com/{bucket}/{path}'

# 안전: Signed URL (15분 TTL)
from google.cloud import storage
blob = bucket.blob(path)
url = blob.generate_signed_url(expiration=timedelta(minutes=15))
return url
```
- [ ] 업로드 후 public URL 대신 Signed URL 반환
- [ ] Signed URL TTL = 15분 (도면 분석 완료 후 만료되도록)

### [RISK-05] Cloud Run Cold Start — 발표 당일 첫 요청 지연
**증상:** 트래픽 없을 때 컨테이너 종료 → 첫 요청 3~5초 지연
**대응:**
```bash
# 발표 30분 전 워밍업 (반드시 실행)
curl https://<cloud-run-url>/health

# 발표 당일 한정 min-instances=1 설정 (미미한 비용)
gcloud run services update interior-api --min-instances=1
# 발표 후 다시 0으로 복구
gcloud run services update interior-api --min-instances=0
```
