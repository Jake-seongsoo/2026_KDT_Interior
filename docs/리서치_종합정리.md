### 1-2. 실패 패턴 — 피해야 할 선례 3가지

**Modsy (2022년 서비스 종료)**
- 원인: 유료 전문 디자이너 커미션 모델 → 비용 증가를 사용자가 부담하지 않으려 함
- 교훈: AI 자동화로 전문가 의존도를 낮추지 않으면 수익 구조가 성립하지 않음

**Laurel & Wolf (재구조화)**
- 원인: 플랫폼이 디자이너를 착취적 수수료로 통제 → 디자이너 이탈 → 서비스 품질 붕괴
- 교훈: 공급자(시공업체, 전문가) 처우가 서비스 품질의 직접 변수

**집닥·숨고 (공정위 시정 2024.12)**
- 원인: "플랫폼은 무책임" 약관으로 소비자 보호 방기 → 정부 제재
- 교훈: "절대 면책"은 법적으로 불가능. 고의·중과실 책임은 반드시 부담해야 함

## 2. 핵심 발견 4가지 — 개발 방향을 바꾸는 인사이트

### 발견 1 — Claude Vision 문·창문 인식률이 임계적으로 낮다

**실측치 (AECV-bench, 2025~2026):**

| 항목 | Claude 3.7 | Gemini 2.5 Pro |
|------|-----------|----------------|
| 전체 평균 | 35% | 41% |
| 침실·욕실 | 70~78% | 70~78% |
| **문(Door)** | **~26%** | **~26%** |
| **창문(Window)** | **~14%** | **~18%** |

**왜 낮은가:**
- 방(Room): 폐쇄 다각형, 텍스트 레이블 → AI가 OCR처럼 인식 가능
- 문: "반원 호선" 기호가 작고 도면마다 크기·방향 다름 → 기하학 패턴 인식 실패
- 창문: "외벽 선이 끊기는 구간"이라는 암묵적 단서에만 의존

**개발 영향:**
- 문·창문·정밀 배치를 목표로 하면 사용자 수정 UI는 필수다.
- 현재 MVP는 보정 UI를 제외하므로, Vision 목표를 **방 이름·방 개수 인식**으로 제한하고 입력 도면도 아파트 분양도면/네이버 부동산 캡처로 좁힌다.
- 문 오인식 → 가구가 문을 막는 배치 생성 → 실제 시공 시 치명적 오류
- 창문 누락 → 채광·환기 무시 가구 배치 → 사용자 신뢰 붕괴

**대응 방법 (상세: `도면분석_기술_리서치.md` 3절):**
1. Few-shot 프롬프팅 (AI Hub 도면 예시 2~3장 첨부) → ~109% 정확도 향상 사례 있음
2. 프롬프트 분해 (공간 인식 → 구조 인식 → 수치 추출 → 문/창문 확인 순서)
3. 낮은 신뢰도 항목 → 사용자 강제 확인 요청 UI

### 발견 2 — 네이버 쇼핑 API 하나로 한샘·일룸·IKEA 모두 커버

**구조:** 한샘·일룸·IKEA는 모두 네이버쇼핑에 공식 입점 → 단일 API 호출로 통합 검색 가능

**MVP 즉시 활용 가능:**
- 일 25,000회 무료 (MVP 예상 사용량 100~500회/일, 50배 여유)
- 상품명·가격(최저~최고)·이미지·구매링크 반환
- 구현 시간: 2~3일

**한 가지 제약 (중요):**
- **재고 상태를 반환하지 않음** — "품절 여부"는 링크 클릭 후 판매처에서만 확인 가능
- 서비스에서 "품절 자동 필터링"을 표방하면 안 됨 (이미 개요서에서 수정 완료)

**FastAPI 구현 코드:** `상품연동_API_리서치.md` 1절 참조

### 발견 3 — AI Hub에 한국 아파트 도면 데이터셋 48,033장 존재

**데이터셋 ID: 71465** (aihub.or.kr)
- 25개 클래스 라벨링: 내력벽·비내력벽·문·창문·거실·침실·욕실·발코니 등
- 형식: PNG 이미지 + 바운딩박스(YOLO) + 세그멘테이션(DeepLabV3+)
- 승인 신청 후 1~3일 이내 다운로드 가능

**활용 방법:**
- Few-shot 예시 도면 선발 (평형별 3~5장)
- Evaluation Set 구성 (KDT 발표 핵심 근거로 활용)
- 정답 JSON 수동 작성 후 v1 → v2 → v3 프롬프트 정확도 비교

**접근 방법 및 데이터 구조:** `도면분석_기술_리서치.md` 6절 참조

### 발견 4 — 공정별 단가 데이터 확보 완료, 즉시 코드화 가능

예산 추정 기능(fc8) 구현에 필요한 서울 기준 단가와 지역 보정계수가 확보됐다.

| 바닥 강마루 | 15~20만원/평 |
| 욕실 기본형 | 300~500만원/개 |
| 욕실 중급형 | 500~800만원/개 |
| 주방 싱크대 | 300~800만원 |

**Python 상수 (UNIT_COSTS, REGIONAL_MULTIPLIER):** `공정별단가_트렌드_리서치.md` 서비스 개발 활용 섹션 참조

**필수 면책 문구 (모든 출력에 포함):**
