## 6. 비용 최적화 전략 (우선순위 순)

### 전략 1 — Imagen Fast 전환 (즉시 적용 가능)
- 적용 전: $0.16/세션 (Standard × 4개 방)
- 적용 후: **$0.08/세션 (Fast × 4개 방)**
- **월 100명 기준: 약 $292 절감 (36% 비용 절감)**
- 트레이드오프: 렌더링 품질 다소 낮음 → A/B 테스트로 사용자 반응 확인 후 결정

### 전략 2 — 방 수 제한 (MVP 기본 적용)
현재 MVP 결정: 톤 6개를 먼저 보여주고, 선택한 톤에 대해서만 주요 방별 이미지 생성

```
방 4개: Imagen × 4 = $0.16/세션
방 3개: Imagen × 3 = $0.12/세션
방 2개: Imagen × 2 = $0.08/세션
```
- 기본 상한은 4개 방으로 두되, 분석된 주요 방이 2~3개면 그만큼 자동 절감
- 욕실·발코니·현관·다용도실은 MVP 이미지 생성 대상에서 제외

### 전략 3 — 프롬프트 캐싱 (즉시 적용)
캐시 대상: 시스템 프롬프트 + Few-shot 예시 도면 (약 4,000 tokens)
- fc2: $0.031 → $0.020/세션 ($0.011 절감)
- fc3: 톤 추천 시스템 프롬프트 일부 절감 (미미)
- **월 100명: 약 $33 절감 (4% 절감)**
- 설정 방법: 시스템 프롬프트를 `cache_control: {"type": "ephemeral"}`로 마킹

### 전략 4 — Web Search 결과 캐싱 (즉시 적용)
- 트렌드 검색 결과를 Redis/메모리에 24시간 캐시
- 같은 트렌드 쿼리가 여러 사용자에게 재사용
- 100명/일 기준 캐시 히트율 60% 가정: $0.01 × 100 × 0.4 × 30 = $12/월 절감 (미미)

### 전략 5 — 사용 제한 (무료 플랜 관리)
- Free: 일 1세션 또는 일 Imagen 4장 제한 → 1세션/일 기본
- Pro: 무제한 (월 ₩9,900 구독)
- 전략: 1일 10회 이상 재추천 사용자 → Pro 유도
- 월 ₩9,900 = $7.5 → 사용자당 API 비용 $0.264 × 10세션 = $2.64이면 4세션 이상부터 흑자

### 전략 6 — Batch API 활용 (비실시간 처리)
