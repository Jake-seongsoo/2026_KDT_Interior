## 1. 실제 토로 → 표면 불만 → 진짜 원인

| 사용자가 실제로 하는 말 (요약) | 표면 불만 | **진짜 원인 (Root)** |
|------|------|------|
| *"300장 저장했는데 막상 무슨 스타일인지 모르겠다"* | 영감이 너무 많다 | **비교 기준을 스스로 못 세움** — 양이 아니라 *판단 축*이 없음 (개요서 P01) |
| 부부가 취향이 달라 조율이 안 됨, "친구 신혼집 보고 와서 또…" | 의견이 안 맞는다 | **객관적 근거가 없어 '취향 대결'이 됨** — 누가 옳은지 판가름할 기준 부재 (homify, 네이트판) |
| *"견적이 가격 비교가 아니라 **구조 비교**", 항목명이 업체마다 달라 비교 불가* | 견적이 헷갈린다 | **정보 비대칭 + 적정가 판단 기준 없음 → 통제감 상실** (개요서 P04, 견적 가이드들) |
| 최저가 미끼·추가비용·자재 등급 사기·역소송… "사기당할 것 같다" | 업체를 못 믿겠다 | **검증 수단이 없는데 비용은 비가역·고액** → 불신이 디폴트값 (lawtalk 사기 3유형, 오늘의집) |
| *"오늘의집 보면 내 집이 초라해 우울해짐"* | SNS가 너무 예쁘다 | **연출된 이미지가 기준점이 됨 + 내 집과 구조가 다름** → 비현실적 기대·상대적 박탈감 (정신의학신문, 대한경제) |
| 셀프 강행 → 예산 1.5배·10개월 임시거주 | 셀프가 힘들다 | **경험 미축적(초보) + 예산·실측 예측 실패** — 일생 1~2번이라 배울 기회가 없음 (개요서 P03, 머니투데이) |

> **핵심 통찰:** 표면 불만은 6가지로 다르지만, 진짜 원인은 거의 한 곳으로 수렴한다 — **"내 선택이 맞다고 확인해줄 객관적 기준·근거가 없다."** 정보가 부족한 게 아니라 *판단 기준*이 없는 것이다.

## 2. 감정 지도 (As-Is 여정 × 감정)

여정 단계마다 솟는 감정과 트리거를 묶었다. **감정 곡선은 2개의 봉우리**(견적 비교 / 계약·시공)에서 정점을 찍는다.

```
감정 강도(부정)
 高 │                                              ▲ 피크2
    │                                             ╱ (계약·시공)
    │                          ▲ 피크1           ╱  불안·사기공포
    │                         ╱ (견적비교)      ╱   후회(셀프)
    │            ┌─ 자기의심  ╱ 답답·불신       ╱
    │  설렘     ╱ 막막·피로 ╱                  ╱
 低 │ ╱────────╱──────────╱──────────────────╱········· (사후) 박탈감·우울
    └──①──────②──────③────④──────⑤──────⑥──────⑦────────▶ 단계
      영감수집  콘셉트   도면·배치 리스트  견적비교  의견검증  계약·시공
```

| 단계 | 솟는 감정 | 트리거 (왜) |
|------|----------|------------|
| **① 영감 수집** | 설렘 → 곧 **막막함·압도감** | 선택지 과잉, 스크랩만 쌓임 |
| **② 콘셉트 결정** | **결정 피로 + 자기 의심**("이게 맞나"), 부부 갈등 | 수렴 기준 부재 → 여기서 자주 멈춤(포기 A) |
| **③ 도면·배치 시뮬** | **불안**(틀릴까) + **답답함**(2D 한계·실측 오차) | 내 공간 시각화 실패 |
| **④ 리스트업** | 상대적 차분(잠깐의 통제감) | 그나마 손에 잡히는 작업 |
| **⑤ 견적 비교** | 🔺**피크1 — 답답함 폭발 + 불신/의심** | 항목 비표준, 적정가 모름 → 지인 추천 회귀(포기 B) |
| **⑥ 의견 검증** | **외로움 + 책임 분산 욕구**, 더 큰 혼란 | 단톡 투표 10:10, 결정 더 미뤄짐 |
| **⑦ 계약·시공** | 🔺**피크2 — 불안 최고조·사기 공포**, (셀프 시) **후회·패닉** | 비가역 고비용, 검증 수단 없음(포기 C) |
| **(사후)** | **박탈감·우울**(SNS 비교) | 연출된 SNS가 기준점, 내 집과 갭 |

## 3. Moodie 관점 시사점

| 감정 봉우리 | 진짜 원인 | Moodie가 끊을 수 있는가 |
|------------|----------|----------------------|
| ②③ 막막·자기의심 | 판단 기준 부재 | ✅ 도면 기반 톤 추천이 "기준"을 제공 — 단 **비교·근거가 약함**(미스핏 A) |
| ⑤ 견적 답답·불신 | 적정가 판단 불가 | ⚠️ **예산 신호 부재**(미스핏 B) — 현재 안 건드림 |
| ⑥ 외로움·검증 욕구 | 결정의 사회화 실패 | ⚠️ 공유는 되나 **의견 회신 경로 없음**(미스핏 A) |
| 사후 박탈감 | SNS 기준점·구조 갭 | ✅ "내 도면 기반"이라는 사실이 완화 — 첫 화면 프레이밍 강화 여지 |

> 감정 지도는 **부정 감정의 정점이 ②(자기의심)·⑤(불신)·⑦(불안)** 에 있음을 보여준다. Moodie는 ②를 정조준(강점)하지만 ⑤·⑦(예산·신뢰)은 비어 있어, JTBD의 "예산 통제감"(미스핏 B)이 다시 우선순위로 확인된다.
